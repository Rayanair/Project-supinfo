# Manuel Utilisateur - SUPCONTENT

Bienvenue sur SUPCONTENT, votre nouveau réseau social dédié au cinéma et aux séries TV.

## 1. Démarrage
### Inscription / Connexion
- Rendez-vous sur la page d'accueil.
- Cliquez sur **"Connexion"** pour accéder à votre compte.
- Si vous n'avez pas de compte, cliquez sur **"S'inscrire"**. Remplissez le formulaire (Email, Pseudo, Mot de passe).
- Une fois connecté, vous accédez à votre **Fil d'actualité**.

## 2. Découvrir des œuvres
### Recherche
- Utilisez la barre de recherche en haut de l'écran.
- Tapez le nom d'un film ou d'une série.
- Cliquez sur un résultat pour voir sa fiche détaillée.

### Fiche Média
Sur la page d'un film ou d'une série, vous pouvez :
- **Voir les détails** : Synopsis, date de sortie, note moyenne.
- **Ajouter à votre bibliothèque** : Cliquez sur les boutons "À voir" ou "Vu".
- **Lire les critiques** : Voyez ce que la communauté en pense.
- **Rédiger une critique** : Cliquez sur "Écrire une critique", donnez une note (étoiles) et partagez votre avis.

## 3. Gérer sa Bibliothèque
- Cliquez sur **"Ma Bibliothèque"** dans le menu.
- Vous y retrouverez toutes vos listes ("À voir", "Terminé", etc.).
- Vous pouvez créer des **Listes Personnalisées** (ex: "Soirée Halloween") via le bouton "Créer une liste".

## 4. Interactions Sociales
### Profil Public
- Cliquez sur le pseudo d'un utilisateur pour voir son profil.
- Vous y verrez ses stats, ses dernières critiques et sa bio.
- Cliquez sur **"Suivre"** pour vous abonner à ses activités.
- Cliquez sur **"Message"** pour discuter en privé.

### Fil d'actualité (Accueil)
- Retrouvez ici les dernières activités de vos amis (critiques, ajouts de films).
- Vous pouvez **Liker** (❤️) ou **Commenter** leurs critiques directement depuis le fil.

## 5. Paramètres & Données
- Cliquez sur votre avatar (en haut à droite) > **Paramètres**.
- **Profil** : Modifiez votre bio, avatar, pseudo.
- **Sécurité** : Changez votre mot de passe.
- **Données** : Téléchargez une copie de toutes vos données (JSON) via le bouton "Exporter".

## 6. Modération (Admin uniquement)
- Si vous êtes administrateur, un lien **"Admin"** apparaît dans le menu.
- Accédez au tableau de bord pour voir les signalements et gérer les utilisateurs.
